alembic migration
1. run "pip install alembic" or add into your requirement.txt file
2. initiate alembic "alembic init alembic"
3. setup migration "alembic revision --autogenerate -m 'Initial migration'" --rev-id "001"
4. run "alembic upgrade head" -> run this when development
5. if run production, setup CI/CD for alembic migration
